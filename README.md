# QuasarDock
Docker files for [Quasar](https://quasar.dev/).

# Dependency
- Docker
- docker-compose

# Get Started

```
# Clone this repository
$ git clone https://github.com/khml/QuasarDock.git

$ cd QuasarDock

# Initialize Quasar App Project
$ ./new.sh YourAppName

# Compose up a container
$ docker-compose up -d

```
